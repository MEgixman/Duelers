Gatsby Documentation Theme

gatsby-theme-documentation is a minimalist Gatsby Theme for docs sites built with MDX and Theme UI.

Installation

yarn add gatsby-theme-documentation

Usage

// gatsby-config.js
module.exports = {
  plugins: [
    'gatsby-theme-documentation'
  ]
}
